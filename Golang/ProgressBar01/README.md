# Golangでプログレスバー

## Install

```
go get github.com/cheggaaa/pb/v3
```

## 実施方法

```
$ go run Golang/ProgressBar01/main.go 
 24527 / 100000 [===============================>--------------------------------------------------------------------------------------------------]  24.53% 01m29s

$ go run Golang/ProgressBar01/main.go 
 100000 / 100000 [==================================================================================================================================] 100.00% 1m59s
The End!
```
